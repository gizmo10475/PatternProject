
# Routes


To simulate a bike:
```
  .../simulate/{bikeid}
```

To stop simulation of a bike:
```
  .../stop/{bikeid}
```

To charge a bike:
```
  .../charge/{bikeid}
```

To stop charging a bike:
```
  .../uncharge/{bikeid}
```

To reset a bike location:
```
  .../resetbike/{bikeid}/{longitude}/{latitude}
```
